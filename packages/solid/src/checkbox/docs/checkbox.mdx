---
id: checkbox
name: Checkbox
description: A checkbox allows users to make a binary choice, i.e. a choice between one of two possible mutually exclusive options.
---

## Import

```ts
import {
  Checkbox,
  CheckboxControl,
  CheckboxInput,
  CheckboxLabel,
} from '@ark-ui/solid'
```

## Usage

The Checkbox component consists of the `CheckboxControl`, `CheckboxInput`, `CheckboxLabel`
components.

```tsx
<Checkbox>
  <CheckboxLabel>Checkbox</CheckboxLabel>
  <CheckboxInput />
  <CheckboxControl />
</Checkbox>
```

### Disabling the Checkbox

To make a checkbox disabled, set the `disabled` prop to `true`

```tsx
<Checkbox disabled>
  <CheckboxLabel>Disabled</CheckboxLabel>
  <CheckboxInput />
  <CheckboxControl />
</Checkbox>
```

### Making the checkbox readonly

To make a checkbox readonly, set the `readOnly` prop to `true`

```tsx
<Checkbox readOnly>
  <CheckboxLabel>Readonly</CheckboxLabel>
  <CheckboxInput />
  <CheckboxControl />
</Checkbox>
```
