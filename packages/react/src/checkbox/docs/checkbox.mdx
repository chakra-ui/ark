---
id: checkbox
name: Checkbox
description: A checkbox allows users to make a binary choice, i.e. a choice between one of two possible mutually exclusive options.
---

## Import

```ts
import {
  Checkbox,
  CheckboxControl,
  CheckboxInput,
  CheckboxLabel,
} from '@ark-ui/react'
```

## Usage

The Checkbox component consists of the `CheckboxControl`, `CheckboxInput`, `CheckboxLabel`
components.

### Plain

```tsx
<Checkbox>
  <CheckboxLabel>Checkbox</CheckboxLabel>
  <CheckboxInput />
  <CheckboxControl />
</Checkbox>
```

### Controlled usage

When the checkbox value changes, the `onChange` callback is invoked.
Pass the checked state via the `checked` prop.

```tsx
const MyCheckbox = () => {
  const [checked, setChecked] = React.useState(false)
  return (
    <Checkbox checked={checked} onChange={(e) => setChecked(e.checked)}>
      <CheckboxLabel>Controlled</CheckboxLabel>
      <CheckboxInput />
      <CheckboxControl />
    </Checkbox>
  )
}
```

### Disabling the Checkbox

To make a checkbox disabled, set the `disabled` prop to `true`

```tsx
<Checkbox disabled>
  <CheckboxLabel>Disabled</CheckboxLabel>
  <CheckboxInput />
  <CheckboxControl />
</Checkbox>
```

### Making the checkbox readonly

To make a checkbox readonly, set the `readOnly` prop to `true`

```tsx
<Checkbox readOnly>
  <CheckboxLabel>Readonly</CheckboxLabel>
  <CheckboxInput />
  <CheckboxControl />
</Checkbox>
```

### Indeterminate

To make a checkbox indeterminate, set the `indeterminate` prop to `true`

```tsx
const DemoIndeterminateCheckbox = () => {
  const [checkedItems, setCheckedItems] = React.useState([false, false])

  const allChecked = checkedItems.every(Boolean)
  const someChecked = checkedItems.some(Boolean) && !allChecked

  return (
    <>
      <Checkbox
        checked={allChecked}
        indeterminate={someChecked}
        onChange={(e) =>
          setCheckedItems([e.checked === true, e.checked === true])
        }
      >
        <CheckboxLabel>Parent Checkbox</CheckboxLabel>
        <CheckboxInput />
        <CheckboxControl />
      </Checkbox>
      <div>
        <Checkbox
          checked={checkedItems[0]}
          onChange={(e) =>
            setCheckedItems([e.checked === true, checkedItems[1]])
          }
        >
          <CheckboxLabel>Child Checkbox 1</CheckboxLabel>
          <CheckboxInput />
          <CheckboxControl />
        </Checkbox>
        <Checkbox
          checked={checkedItems[1]}
          onChange={(e) =>
            setCheckedItems([checkedItems[0], e.checked === true])
          }
        >
          <CheckboxLabel>Child Checkbox 2</CheckboxLabel>
          <CheckboxInput />
          <CheckboxControl />
        </Checkbox>
      </div>
    </>
  )
}
```
