# Checkbox

A checkbox allows users to make a binary choice, i.e. a choice between one of two possible mutually
exclusive options.

## Import

```ts
import { Checkbox, CheckboxControl, CheckboxInput, CheckboxLabel } from '@ark-ui/react'
```

## Usage

The Checkbox component consists of the `CheckboxControl`, `CheckboxInput`, `CheckboxLabel`
components.

### Plain

```tsx
<Checkbox>
  <CheckboxLabel>Checkbox</CheckboxLabel>
  <CheckboxInput />
  <CheckboxControl />
</Checkbox>
```

### Indeterminate

To make a checkbox indeterminate, set the `indeterminate` prop to `true`

```tsx
<Checkbox indeterminate>
  <CheckboxLabel>Indeterminate</CheckboxLabel>
  <CheckboxInput />
  <CheckboxControl />
</Checkbox>
```

### Disabling the Checkbox

To make a checkbox disabled, set the `disabled` prop to `true`

```tsx
<Checkbox disabled>
  <CheckboxLabel>Disabled</CheckboxLabel>
  <CheckboxInput />
  <CheckboxControl />
</Checkbox>
```

### Making the checkbox readonly

To make a checkbox readonly, set the `readonly` prop to `true`

```tsx
<Checkbox readonly>
  <CheckboxLabel>Readonly</CheckboxLabel>
  <CheckboxInput />
  <CheckboxControl />
</Checkbox>
```
