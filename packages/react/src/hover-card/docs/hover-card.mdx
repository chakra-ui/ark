# Hover Card

A hover card allows sighted users to preview content available behind a link.

## Import

```ts
import {
  HoverCard,
  HoverCardArrow,
  HoverCardContent,
  HoverCardInnerArrow,
  HoverCardPortal,
  HoverCardPositioner,
  HoverCardTrigger,
} from '@ark-ui/react'
```

## Usage

The HoverCard component consists of the `HoverCardArrow`, `HoverCardContent`, `HoverCardInnerArrow`,
`HoverCardPortal`, `HoverCardPositioner` and `HoverCardTrigger` components.

Note that the `HoverCardTrigger` component accepts a single JSX element which will receive all
necessary props to add the event listeners and attributes to ensure the desired accessibility.

### Plain

```tsx
<HoverCard>
  <HoverCardTrigger>
    <a href="https://zagjs.com" target="_blank" rel="noreferrer">
      zag
    </a>
  </HoverCardTrigger>

  <HoverCardPortal>
    <HoverCardPositioner>
      <HoverCardContent>
        <HoverCardArrow>
          <HoverCardInnerArrow />
        </HoverCardArrow>
        zag Preview
      </HoverCardContent>
    </HoverCardPositioner>
  </HoverCardPortal>
</HoverCard>
```
