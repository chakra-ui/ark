## Import

```ts
import {
  Tooltip,
  TooltipArrow,
  TooltipContent,
  TooltipInnerArrow,
  TooltipPositioner,
  TooltipTrigger,
} from '@ark-ui/react'
```

## Usage

The Tooltip component consists of the `TooltipTrigger`, `TooltipPositioner`, `TooltipArrow`,
`TooltipInnerArrow` and `TooltipContent` components. Combine them as desired to fit your design
system.

Note that the `TooltipTrigger` component accepts a single JSX element which will receive all
necessary props to add the event listeners and attributes to ensure the desired accessibility.

### Plain

```tsx
<Tooltip>
  <TooltipTrigger>
    <span>hover me</span>
  </TooltipTrigger>

  <TooltipPositioner>
    <TooltipContent>My tooltip content</TooltipContent>
  </TooltipPositioner>
</Tooltip>
```

### Adding an arrow

To render an arrow within the tooltip render the components `TooltipArrow` and `TooltipInnerArrow`
within the `TooltipPositioner`.

```tsx {7-9}
<Tooltip>
  <TooltipTrigger>
    <span>hover me</span>
  </TooltipTrigger>

  <TooltipPositioner>
    <TooltipArrow>
      <TooltipInnerArrow />
    </TooltipArrow>

    <TooltipContent>My tooltip content</TooltipContent>
  </TooltipPositioner>
</Tooltip>
```

### Disabled

To disable showing a tooltip, pass the `disabled` prop to the Tooltip component.

```tsx
<Tooltip disabled>{/*...*/}</Tooltip>
```

### Changing the placement

Control the placement, gutter and offset of the tooltip with the `positioning` prop.

```tsx
<Tooltip positioning={{ placement: 'left-start', gutter: 16, offset: 12 }}>
  {/*...*/}
</Tooltip>
```

### Timings

By default, the tooltip is designed to open after `1000ms` and close after `500ms`. You can
customize this by passing the `openDelay` and `closeDelay` props.

```tsx
<Tooltip openDelay={500} closeDelay={200}>
  {/*...*/}
</Tooltip>
```

### Custom Component

When using a custom component as trigger for the tooltip, ensure that the component forwards all
necessary DOM attributes.

```tsx
const CustomComponent = (props) => <span {...props}>hover me</span>
```

```tsx {3}
<Tooltip>
  <TooltipTrigger>
    <CustomComponent />
  </TooltipTrigger>

  <TooltipPositioner>
    <TooltipContent>My tooltip content</TooltipContent>
  </TooltipPositioner>
</Tooltip>
```
