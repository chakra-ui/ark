## Anatomy

To set up the hover card correctly, you'll need to understand its anatomy and
how we name its parts.

> Each part includes a `data-part` attribute to help identify them in the DOM.

## Data Attributes

### Root

| Name       | Type         | Description                          | Default Value |
|------------|--------------|--------------------------------------|---------------|
| data-scope | 'hover-card' | Indicates the scope of this part     | -             |
| data-part  | 'root'       | Indicates which part is this element | -             |

### Trigger

| Name           | Type                                                                                                                                                               | Description                                         | Default Value |
|----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------|---------------|
| data-scope     | 'hover-card'                                                                                                                                                       | Indicates the scope of this part                    | -             |
| data-part      | 'trigger'                                                                                                                                                          | Indicates which part is this element                | -             |
| data-state     | 'open' \| 'closed'                                                                                                                                                 | Indicates the current state of this element         | -             |
| data-placement | 'bottom' \| 'bottom-end' \| 'bottom-start' \| 'left' \| 'left-end' \| 'left-start' \| 'top' \| 'top-end' \| 'top-start' \| 'right' \| 'right-end' \| 'right start' | Indicates the preferred placement of the hover card | -             |
