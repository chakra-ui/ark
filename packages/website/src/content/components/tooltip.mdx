we name its parts.

## Data Attributes

### Root

| Name       | Type      | Description                          | Default Value |
|------------|-----------|--------------------------------------|---------------|
| data-scope | 'tooltip' | Indicates the scope of this part     | -             |
| data-part  | 'root'    | Indicates which part is this element | -             |

### Positioner

| Name       | Type         | Description                          | Default Value |
|------------|--------------|--------------------------------------|---------------|
| data-scope | 'tooltip'    | Indicates the scope of this part     | -             |
| data-part  | 'positioner' | Indicates which part is this element | -             |

### Arrow

| Name       | Type      | Description                          | Default Value |
|------------|-----------|--------------------------------------|---------------|
| data-scope | 'tooltip' | Indicates the scope of this part     | -             |
| data-part  | 'arrow'   | Indicates which part is this element | -             |

### Arrow Tip

| Name       | Type        | Description                          | Default Value |
|------------|-------------|--------------------------------------|---------------|
| data-scope | 'tooltip'   | Indicates the scope of this part     | -             |
| data-part  | 'arrow-tip' | Indicates which part is this element | -             |

### Content

| Name       | Type               | Description                                | Default Value |
|------------|--------------------|--------------------------------------------|---------------|
| data-scope | 'tooltip'          | Indicates the scope of this part           | -             |
| data-part  | 'content'          | Indicates which part is this element       | -             |
| data-state | 'open' \| 'closed' | Indicates the current state of the content | -             |
