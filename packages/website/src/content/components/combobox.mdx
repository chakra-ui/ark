## Data Attributes

### Root

| Name       | Type       | Description                          | Default Value |
|------------|------------|--------------------------------------|---------------|
| data-scope | 'combobox' | Indicates the scope of this part     | -             |
| data-part  | 'root'     | Indicates which part is this element | -             |

### Label

| Name        | Type       | Description                              | Default Value |
|-------------|------------|------------------------------------------|---------------|
| data-scope  | 'combobox' | Indicates the scope of this part         | -             |
| data-part   | 'label'    | Indicates which part is this element     | -             |
| data-focus? | boolean    | Present when this element receives focus | false         |

### Control
