## Usage

```tsx live story=Basic
<Accordion.Root defaultValue={['React']}>
  {['React', 'Solid', 'Vue'].map((item, id) => (
    <Accordion.Item key={id} value={item}>
      <Accordion.ItemTrigger>
        {item}
        <Accordion.ItemIndicator>
          <ChevronDownIcon />
        </Accordion.ItemIndicator>
      </Accordion.ItemTrigger>
      <Accordion.ItemContent>
        <div>
          {item} is a JavaScript library focused on building user interfaces.
        </div>
      </Accordion.ItemContent>
    </Accordion.Item>
  ))}
</Accordion.Root>
```

## Examples

### Collapsible Panels

To create a collapsible Accordion where all panels can be closed simultaneously,
utilize the `collapsible` prop:

```tsx live story=Collapsible
<Accordion.Root defaultValue={['React']} collapsible>
  {['React', 'Solid', 'Vue'].map((item, id) => (
    <Accordion.Item key={id} value={item}>
      <Accordion.ItemTrigger>
        {item}
        <Accordion.ItemIndicator>
          <ChevronDownIcon />
        </Accordion.ItemIndicator>
      </Accordion.ItemTrigger>
      <Accordion.ItemContent>
        <div>{item} is a JavaScript library for building user interfaces.</div>
      </Accordion.ItemContent>
    </Accordion.Item>
  ))}
</Accordion.Root>
```

### Multiple Panels Open

For an Accordion that allows keeping multiple panels open, apply the `multiple`
prop:

```tsx live story=Multiple
<Accordion.Root defaultValue={['React']} multiple>
  {['React', 'Solid', 'Vue'].map((item, id) => (
    <Accordion.Item key={id} value={item}>
      <Accordion.ItemTrigger>
        {item}
        <Accordion.ItemIndicator>
          <ChevronDownIcon />
        </Accordion.ItemIndicator>
      </Accordion.ItemTrigger>
      <Accordion.ItemContent>
        <div>{item} is a JavaScript library for building user interfaces.</div>
```
