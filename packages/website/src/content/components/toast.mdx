## Anatomy

To set up the toast correctly, you'll need to understand its anatomy and how we
name its parts.

> Each part includes a `data-part` attribute to help identify them in the DOM.

<Anatomy id="toast" />

## Examples

To use the Toast component in your application, you need to set it up using the
`createToaster` hook. This hook manages the placement and grouping of toasts,
and provides a `toast` object needed to create toast notification.

To create a basic Toast, use the `toast.create` method to display a
notification.

## Data Attributes

### Group

| Name           | Type                                                                                                                                                               | Description                                          | Default Value |
|----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------|---------------|
| data-scope     | 'toast'                                                                                                                                                            | Indicates the scope of this part                     | -             |
| data-part      | 'group'                                                                                                                                                            | Indicates which part is this element                 | -             |
| data-placement | 'bottom' \| 'bottom-end' \| 'bottom-start' \| 'left' \| 'left-end' \| 'left-start' \| 'top' \| 'top-end' \| 'top-start' \| 'right' \| 'right-end' \| 'right start' | Specifies the placement strategy for the group       | -             |
| data-side      | 'top' \| 'left' \| 'bottom' \| 'right'                                                                                                                             | Specifies which side of the viewport the group is on | -             |
| data-align     | 'start' \| 'center' \| 'end'                                                                                                                                       | Aligns the group within its side                     | -             |

### Root

| Name           | Type                                                                                                                                                               | Description                                          | Default Value |
|----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------|---------------|
| data-scope     | 'toast'                                                                                                                                                            | Indicates the scope of this part                     | -             |
| data-part      | 'root'                                                                                                                                                             | Indicates which part is this element                 | -             |
| data-placement | 'bottom' \| 'bottom-end' \| 'bottom-start' \| 'left' \| 'left-end' \| 'left-start' \| 'top' \| 'top-end' \| 'top-start' \| 'right' \| 'right-end' \| 'right start' | Specifies the placement strategy for the toast       | -             |
| data-side      | 'top' \| 'left' \| 'bottom' \| 'right'                                                                                                                             | Specifies which side of the viewport the toast is on | -             |
| data-align     | 'start' \| 'center' \| 'end'                                                                                                                                       | Aligns the toast within its side                     | -             |
| data-first     | boolean                                                                                                                                                            | Whether this is the first toast in the group         | -             |
| data-last      | boolean                                                                                                                                                            | Whether this is the last toast in the group          | -             |
| data-stack     | boolean                                                                                                                                                            | Whether the toast is part of a stack                 | -             |
| data-mounted   | boolean                                                                                                                                                            | Whether the toast is currently mounted               | -             |
| data-paused    | boolean                                                                                                                                                            | Whether the toas is paused                           | -             |
