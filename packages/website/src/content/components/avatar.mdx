## Data Attributes

Explore all the data-attributes for this component below:

### Root

| Name | Type | Description | Default Value |
| ---- | ---- | ----------- | ------------- |
| data-scope | 'avatar' | Indicates the scope of this part | - |
| data-part | 'root' | Indicates which part is this element | - |

### Fallback

| Name | Type | Description | Default Value |
| ---- | ---- | ----------- | ------------- |
| data-scope | 'avatar' | Indicates the scope of this part | - |
| data-part | 'fallback' | Indicates which part is this element | - |
| data-state | 'hidden' \| 'visible' | Indicates the current state of this element | - |

### Image

| Name | Type | Description | Default Value |
| ---- | ---- | ----------- | ------------- |
| data-scope | 'avatar' | Indicates the scope of this part | - |
| data-part | 'image' | Indicates which part is this element | - |
| data-state | 'hidden' \| 'visible' | Indicates the current state of this element | - |
