## Anatomy

To set up the radio group correctly, you'll need to understand its anatomy and
how we name its parts.

> Each part includes a `data-part` attribute to help identify them in the DOM.

## Data Attributes

### Root

| Name             | Type                       | Description                                 | Default Value |
|------------------|----------------------------|---------------------------------------------|---------------|
| data-scope       | 'radio-group'              | Indicates the scope of this part            | -             |
| data-part        | 'root'                     | Indicates which part is this element        | -             |
| data-orientation | 'horizontal' \| 'vertical' | Reflects the orientation of the radio group | 'vertical'    |

### Item

| Name             | Type                       | Description                                 | Default Value |
|------------------|----------------------------|---------------------------------------------|---------------|
| data-scope       | 'radio-group'              | Indicates the scope of this part            | -             |
| data-part        | 'item'                     | Indicates which part is this element        | -             |
| data-state       | 'checked' \| 'unchecked'   | Indicates the current state of the item     | -             |
| data-orientation | 'horizontal' \| 'vertical' | Reflects the orientation of the radio group | -             |

### Control
