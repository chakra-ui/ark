## Data Attributes

### Root

| Name             | Type                       | Description                                  | Default Value |
|------------------|----------------------------|----------------------------------------------|---------------|
| data-scope       | 'toggle-group'             | Indicates the scope of this part             | -             |
| data-part        | 'root'                     | Indicates which part is this element         | -             |
| data-orientation | 'horizontal' \| 'vertical' | Reflects the orientation of the toggle group | -             |

### Item

| Name             | Type                       | Description                                  | Default Value |
|------------------|----------------------------|----------------------------------------------|---------------|
| data-scope       | 'toggle-group'             | Indicates the scope of this part             | -             |
| data-part        | 'item'                     | Indicates which part is this element         | -             |
| data-ownedby     | string                     | Indicates the id of the owning root element  | -             |
| data-orientation | 'horizontal' \| 'vertical' | Reflects the orientation of the toggle group | -             |
| data-state       | 'on' \| 'off'              | Indicates the current state of the toggle    | -             |
