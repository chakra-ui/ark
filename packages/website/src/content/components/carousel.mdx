## Data Attributes

### Root

| Name             | Type                       | Description                              | Default Value |
|------------------|----------------------------|------------------------------------------|---------------|
| data-scope       | 'carousel'                 | Indicates the scope of this part         | -             |
| data-part        | 'root'                     | Indicates which part is this element     | -             |
| data-orientation | 'horizontal' \| 'vertical' | Reflects the current display orientation | 'horizontal'  |
