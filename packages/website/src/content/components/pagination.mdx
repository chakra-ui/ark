## Data Attributes

### Root

| Name       | Type         | Description                          | Default Value |
|------------|--------------|--------------------------------------|---------------|
| data-scope | 'pagination' | Indicates the scope of this part     | -             |
| data-part  | 'root'       | Indicates which part is this element | -             |

### Prev Trigger

| Name       | Type           | Description                          | Default Value |
|------------|----------------|--------------------------------------|---------------|
| data-scope | 'pagination'   | Indicates the scope of this part     | -             |
| data-part  | 'prev-trigger' | Indicates which part is this element | -             |

### Next Trigger

| Name       | Type           | Description                          | Default Value |
|------------|----------------|--------------------------------------|---------------|
| data-scope | 'pagination'   | Indicates the scope of this part     | -             |
| data-part  | 'next-trigger' | Indicates which part is this element | -             |

### Item

| Name       | Type         | Description                          | Default Value |
|------------|--------------|--------------------------------------|---------------|
| data-scope | 'pagination' | Indicates the scope of this part     | -             |
| data-part  | 'item'       | Indicates which part is this element | -             |
| data-index | number       | Specifies the index of the item      | -             |

### Ellipsis

| Name       | Type         | Description                          | Default Value |
|------------|--------------|--------------------------------------|---------------|
| data-scope | 'pagination' | Indicates the scope of this part     | -             |
| data-part  | 'ellipsis'   | Indicates which part is this element | -             |
