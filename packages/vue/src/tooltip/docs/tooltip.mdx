---
id: tooltip
name: Tooltip
description:
  A tooltip is a brief, informative message that appears when a user interacts
  with an element. Tooltips are usually initiated when a button is focused or
  hovered.
---

## Import

```ts
import {
  Tooltip,
  TooltipArrow,
  TooltipArrowTip,
  TooltipContent,
  TooltipPositioner,
  TooltipTrigger,
} from '@ark-ui/vue'
```

## Usage

The Tooltip component consists of the `TooltipTrigger`, `TooltipPositioner`,
`TooltipArrow`, `TooltipArrowTip` and `TooltipContent` components. Combine them
as desired to fit your design system.

Note that the `TooltipTrigger` component accepts a single JSX element which will
receive all necessary props to add the event listeners and attributes to ensure
the desired accessibility.

```vue
<script setup lang="ts">
import {
  Tooltip,
  TooltipContent,
  TooltipPositioner,
  TooltipTrigger,
} from '@ark-ui/vue'
</script>
<template>
  <Tooltip>
    <TooltipTrigger>
      <span>hover me</span>
    </TooltipTrigger>

    <TooltipPositioner>
      <TooltipContent>My tooltip content</TooltipContent>
    </TooltipPositioner>
  </Tooltip>
</template>
```

### Adding an arrow

To render an arrow within the tooltip render the components `TooltipArrow` and
`TooltipArrowTip` within the `TooltipPositioner`.

```vue {7-9}
<script setup lang="ts">
import {
  Tooltip,
  TooltipArrow,
  TooltipArrowTip,
  TooltipContent,
  TooltipPositioner,
  TooltipTrigger,
} from '@ark-ui/vue'
</script>
<template>
  <Tooltip>
    <TooltipTrigger>
      <span>hover me</span>
    </TooltipTrigger>

    <TooltipPositioner>
      <TooltipArrow>
        <TooltipArrowTip />
      </TooltipArrow>

      <TooltipContent>My tooltip content</TooltipContent>
    </TooltipPositioner>
  </Tooltip>
</template>
```

### Disabled

To disable showing a tooltip, pass the `disabled` prop to the Tooltip component.

```vue
<script setup lang="ts">
import { Tooltip } from '@ark-ui/vue'
</script>
<template>
  <Tooltip disabled>
    <!-- ... -->
  </Tooltip>
</template>
```

### Changing the placement

Control the placement, gutter and offset of the tooltip with the `positioning`
prop.

```vue
<script setup lang="ts">
import { Tooltip } from '@ark-ui/vue'
</script>
<template>
  <Tooltip
    :positioning="{
      placement: 'left-start',
      gutter: 16,
      offset: { mainAxis: 12 },
    }"
  >
    <!-- ... -->
  </Tooltip>
</template>
```

### Timings

By default, the tooltip is designed to open after `1000ms` and close after
`500ms`. You can customize this by passing the `openDelay` and `closeDelay`
props.

```vue
<script setup lang="ts">
import { Tooltip } from '@ark-ui/vue'
</script>
<template>
  <Tooltip :open-delay="500" :close-delay="200">
    <!-- ... -->
  </Tooltip>
</template>
```

### Custom Component

When using a custom component as trigger for the tooltip, ensure that the
component forwards all necessary DOM attributes.
