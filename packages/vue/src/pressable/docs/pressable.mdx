---
id: pressable
name: Pressable
description:
  Pressable handles press interactions across mouse, touch, keyboard, and screen
  readers.
---

## Import

```ts
import { Pressable } from '@ark-ui/vue'
```

## Usage

The Pressable component consists only of the `Pressable` component.

```vue
<script setup lang="ts">
import { Pressable } from '@ark-ui/vue'

const handlePress = () => alert('onPress')
</script>
<template>
  <Pressable @press="handlePress">Press me</Pressable>
</template>
```

### Long Press

`@long-press` is called when the element has been pressed for 500 milliseconds.

```vue
<script setup lang="ts">
import { Pressable } from '@ark-ui/vue'

const handleLongPress = () => alert('onPress')
</script>
<template>
  <Pressable @long-press="handlePress">Press me long</Pressable>
</template>
```

### Disabled

To disable a pressable, set the `disabled` prop to `true`.

```vue
<script setup lang="ts">
import { Pressable } from '@ark-ui/vue'
</script>
<template>
  <Pressable isDisabled>Disabled</Pressable>
</template>
```
