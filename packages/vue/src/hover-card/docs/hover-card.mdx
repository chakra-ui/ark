---
id: hover-card
name: Hover Card
description:
  A hover card allows sighted users to preview content available behind a link.
---

## Import

```ts
import {
  HoverCard,
  HoverCardArrow,
  HoverCardContent,
  HoverCardInnerArrow,
  HoverCardPositioner,
  HoverCardTrigger,
  Portal,
} from '@ark-ui/vue'
```

## Usage

The HoverCard component consists of the `HoverCardArrow`, `HoverCardContent`,
`HoverCardInnerArrow`, `HoverCardPositioner` and `HoverCardTrigger` components.

Note that the `HoverCardTrigger` component accepts a single JSX element which
will receive all necessary props to add the event listeners and attributes to
ensure the desired accessibility.

```vue
<script setup lang="ts">
import { Teleport } from 'vue'
import {
  HoverCard,
  HoverCardArrow,
  HoverCardContent,
  HoverCardInnerArrow,
  HoverCardPositioner,
  HoverCardTrigger,
  Portal,
} from '@ark-ui/vue'
</script>
<template>
  <HoverCard>
    <HoverCardTrigger>
      <a href="https://zagjs.com" target="_blank" rel="noreferrer"> zag </a>
    </HoverCardTrigger>

    <Teleport to="body">
      <HoverCardPositioner>
        <HoverCardContent>
          <HoverCardArrow>
            <HoverCardInnerArrow />
          </HoverCardArrow>
          zag Preview
        </HoverCardContent>
      </HoverCardPositioner>
    </Teleport>
  </HoverCard>
</template>
```
