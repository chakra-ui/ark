---
id: radio-group
name: Radio Group
description:
  A radio group allows users to make a single choice from a selected number of
  options.
---

## Import

```ts
import {
  RadioGroup,
  Radio,
  RadioControl,
  RadioGroupLabel,
  RadioInput,
  RadioLabel,
} from '@ark-ui/vue'
```

## Usage

The RadioGroup component consists of the `RadioControl`, `Radio`,
`RadioGroupLabel`, `RadioInput`, and `RadioLabel` components. Combine them as
desired to fit your design system.

```vue
<script setup lang="ts">
import {
  RadioGroup,
  Radio,
  RadioControl,
  RadioGroupLabel,
  RadioInput,
  RadioLabel,
} from '@ark-ui/vue'

const options = [
  { id: 'apple', label: 'Apples' },
  { id: 'orange', label: 'Oranges' },
  { id: 'mango', label: 'Mangoes' },
  { id: 'grape', label: 'Grapes' },
]
</script>
<template>
  <RadioGroup>
    <RadioGroupLabel>Fruits</RadioGroupLabel>
    <Radio v-for="(option, id) in options" :key="id" :value="option.id">
      <RadioLabel>{{ option.label }}</RadioLabel>
      <RadioInput />
      <RadioControl />
    </Radio>
  </RadioGroup>
</template>
```

### Disabling the radio group

To make a radio group disabled, set the `disabled` prop to `true`.

```vue
<script setup lang="ts">
import { RadioGroup } from '@ark-ui/vue'
</script>
<template>
  <RadioGroup disabled>
    <!-- ... -->
  </RadioGroup>
</template>
```

### Making the radio group readonly

To make a radio group readonly, set the `readOnly` prop to `true`.

```vue
<script setup lang="ts">
import { RadioGroup } from '@ark-ui/vue'
</script>
<template>
  <RadioGroup readOnly>
    <!-- ... -->
  </RadioGroup>
</template>
```

### Setting the initial value

To set the radio group's initial value, use the `v-model` directive to the value
of the radio item to be selected by default.

```vue
<script setup lang="ts">
import { RadioGroup } from '@ark-ui/vue'
const currentValue = ref('apple')
</script>
<template>
  <RadioGroup v-model="currentValue">
    <!-- ... -->
  </RadioGroup>
</template>
```

### Listening for changes

When the radio group value changes, the `@change` callback is invoked.

```vue
<script setup lang="ts">
import { RadioGroup } from '@ark-ui/vue'
const handleOnChange = (details) => {
  console.log(details.value)
}
</script>
<template>
  <RadioGroup @change="handleOnChange">
    <!-- ... -->
  </RadioGroup>
</template>
```
