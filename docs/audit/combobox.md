## Summary
- Status: Public API, accessibility, and behavior parity with React for the root, trigger, input, and empty parts. Storybook coverage expanded from 5 to 13 Angular stories (out of 17 React stories); only four React stories that depend on missing Angular helpers (`Highlight`, `useAsyncList`/`@tanstack/react-virtual`, `Combobox.Context` render-prop with item-level access, and the `flushSync` / list-collection helpers used by `Creatable` and `RehydrateValue`) remain deferred.
- Highest-risk gaps: None for the public API. Remaining gaps are story-only coverage that requires upstream Angular helpers (highlight component, async-list/virtualizer integration) and a render-prop ergonomic for in-item context reads.
