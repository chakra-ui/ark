# Toast

The toast component is used to give feedback to users after an action has taken place.

## Usage

### Create the `AppToastProvider`

Use the toast components to compose a `ToastProvider` for your application. The granular component
structure allows access to every DOM element.

import AppToastProviderReact from '!!raw-loader!./code/AppToastProvider.react'
import AppToastProviderSolid from '!!raw-loader!./code/AppToastProvider.solid'

<Tabs groupId="framework">
  <TabItem value="react" label="React">
    <CodeBlock language="tsx" children={AppToastProviderReact} />
  </TabItem>
  <TabItem value="solid" label="SolidJS">
    <CodeBlock language="tsx" children={AppToastProviderSolid} />
  </TabItem>
</Tabs>

### Wrap your App with the `AppToastProvider`

Wrapping the application provides all child components access to the toast context with the
`useToast` hook.

import wrapAppToastProviderReact from '!!raw-loader!./code/wrapAppToastProvider.react'
import wrapAppToastProviderSolid from '!!raw-loader!./code/wrapAppToastProvider.solid'

<Tabs groupId="framework">

  </TabItem>

    <CodeBlock language="tsx" children={wrapAppToastProviderSolid} />
  </TabItem>
</Tabs>

### Create toasts with `useToast`

The `useToast` hook is your _central toast intelligence unit_ to control the toasts in your
application. Create, update, remove and upsert toasts with the returned `toast` instance.

import createToastSolid from '!!raw-loader!./code/createToast.solid'

    <CodeBlock language="tsx" children={createToastSolid} />

</Tabs>
