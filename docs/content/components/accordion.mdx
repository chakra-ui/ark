## Usage

Here is how you use this with multiple

```jsx
// [!code highlight:2]
<Accordion.Root multiple>
  <Accordion.Item value="item-1">
    …
  </Accordion.Item>
  <Accordion.Item value="item-2">
    …
  </Accordion.Item>
</AccordionRoot>
```
